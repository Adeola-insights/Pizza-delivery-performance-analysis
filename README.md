# Pizza-Delivery-Performance-Analysis
An Excel-based data analysis project exploring pizza delivery performance using pivot tables, KPIs, and an interactive dashboard.
# Project Overview
This project analyzes pizza delivery operations to uncover insights about order volume, delivery performance, customer preferences, and operational efficiency.
Using Excel (Pivot Tables, KPIs, and Dashboards), the analysis transforms raw delivery data into actionable business insights.

The goal is to demonstrate data cleaning, analysis, visualization, and storytelling using Excel.

# Business Objectives
To help pizza delivery businesses:
1. Understand ordering patterns
2. Identify top-performing restaurants and locations
3. Evaluate delivery efficiency
4. Improve customer experience through data-driven decisions
# Business Questions
This analysis answers the following questions:
* How many total orders were placed?
* How many restsurants are operating?
* Which restaurants recieve the most orders?
* Which location generate the highest demand?
* What pizza types and sizes are most popular?
* How pizza traffic and delay affect delivery time?
* Which payment methods are preferred by customers?
 # Key Performance Indicators (KPIs)
 The dashboard tracks the following KPIs:
 - Total orders
  - Total Restaurants
  - Average Delivery Time (minutes)
  - Average Delivery Delay (minutes)
    These KPIs provide a high level snapshot of delivery performance.

# Dashboard Overview
The Excel dashboard combines KPIs, charts, and slicers to create an interactive and visually clear summary of insights.
Dashboard Features:
* KPI cards for quick performance tracking
* Bar charts for comparisons
* Donut chart for pizza size distribution
* Monthly trend analysis

# Key Insights 
* A small number of restaurants generate the majority of orders.
* Medium-sized pizzas are the most popular choice.
* Some locations consistently outperform others in order volume.
* Average delivery time is reasonable, but delays still exist.
* Customers strongly prefer digital payment methods.
# Business Recommendations
* Allocate more delivery resources to high-demand locations.
* Support top-performing restaurants with better logistics.
* Monitor delivery delays during peak periods.
* Use popular pizza types and sizes for targeted promotions.
* Continue promoting digital payment options to improve efficiency.
